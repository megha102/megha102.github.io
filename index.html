Consumer Debt Analysis

This project explores U.S. consumer debt repayment behavior using Machine Learning and Time Series Analysis. It predicts repayment likelihood within five years of charge-off and forecasts financial trends pre- and post-COVID. The analysis includes **classification models**, **time-series forecasting**, and **interactive Tableau visualizations** to examine regional and macroeconomic factors. The project is tested on Windows (with AzureML compatibility issues on M1/M2 silicon chips).
